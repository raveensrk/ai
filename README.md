# ai
Repository to mess around with AI things.

## Agent Skills

Skills follow the [Agent Skills](https://agentskills.io) protocol and live
locally in this repo under [.agents/skills/](.agents/skills/) — one directory
per skill, each containing a `SKILL.md`.

Current skills:

- [reverse-image-search-video](.agents/skills/reverse-image-search-video/SKILL.md)

To add a skill, create a new directory under [.agents/skills/](.agents/skills/)
with its own `SKILL.md`.
